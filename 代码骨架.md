## 2. 数据流(两阶段)

```
prepare_data.py
   原始图 ──圆裁剪+归一化──► 预处理图(缓存到 data_processed/)
                          └─ splits.py: 跨集去重 + 患者级分层K折 ──► folds.csv

Stage-1 (stage1_pretrain.py)
   EyePACS+APTOS+Messidor-2 (DR 0–4) ─► ConvNeXt-Tiny + 临时5类头 ─► backbone.pth

Stage-2 (stage2_finetune.py)
   load backbone.pth (丢弃5类头)
   原始 Messidor (DR 0–3 + ME 0–2) ─► MultiTaskNet ─► best_qwk.pth

evaluate.py / run_gradcam.py
   best_qwk.pth ─► QWK/混淆矩阵 ; Grad-CAM 热力图
```

## 4. 配置 schema(stage2_finetune.yaml,10G 显存友好默认)

```yaml
seed: 42
data:
  root: data_processed/messidor
  folds_csv: folds.csv
  fold: 0
  image_size: 512            # OOM 时降到 448 / 384
model:
  backbone: convnext_tiny
  pretrained_imagenet: true
  backbone_ckpt: checkpoints/stage1/backbone.pth   # 为空则走 ImageNet(Stage-1 跳过时)
  grad_checkpoint: true      # 10G 关键:用算力换显存,保住 512px
  attention: cross           # none | specific | cross(=完整CANet式)
  head: corn                 # corn | coral | softmax(三者消融)
  num_classes: {dr: 4, me: 3}
loss:
  type: ldam                 # ce | focal | ldam
  drw_defer_epoch: 30
  task_weights: {dr: 1.0, me: 1.0}   # 或 uncertainty 自动
train:
  epochs: 80
  batch_size: 8              # 10G 默认;配合 accum 等效 16
  grad_accum: 2              # 梯度累积,等效 batch = batch_size*grad_accum
  optimizer: adamw
  lr: 1.0e-4
  weight_decay: 0.05
  amp: true                  # 混合精度,激活减半
  channels_last: true
  early_stop_metric: qwk_mean       # 以 (qwk_dr+qwk_me)/2 早停
eval:
  tta: false
```

> **10G 显存阶梯**:先开 `amp`+`grad_checkpoint`;仍 OOM 则降 `batch_size`(8→4)并提 `grad_accum`;再不行降 `image_size`(512→448→384)。真实占用需在目标卡上跑一个 step 实测。
>
> **消融**通过改 `model.attention` / `model.head` / `loss.type` 一键切换,对应技术路线 §7 消融表。`head` 同时记录 `corn` / `coral` / `softmax` 三者 QWK。

## 5. 已确认的设计决策

1. **包名** ✅ 用 `drnet/`。
2. **主干来源** ✅ 用 `timm` 的 `convnext_tiny`(自带 ImageNet 预训练),不复制官方 ConvNeXt 代码。
3. **序数头** ✅ 默认 **CORN**,但 `head` 配置项同时支持 `coral` / `softmax`,**baseline 阶段三者都跑、记录 QWK 做消融**。
   - 计算差异(已与你确认):CORAL 共享投影向量、仅偏置不同,结构上保证单调一致,每样本参与每个二分类;CORN 各阈值独立权重,用条件概率链 `P(y>k)=Π f_j` 保证一致,但靠后阈值只在条件子集上训练。本项目阈值少(DR 3、ME 2),CORN 子集碎片化不严重故默认 CORN;若 R3/M2 在 CORN 下方差大则退回 CORAL(全样本训每任务、对极稀有等级更稳)。
4. **Stage-1 可跳过** ✅ 做成开关:`configs/stage2_finetune.yaml` 的 `model.backbone_ckpt` 留空即走 ImageNet 权重。算力紧张时先跳过 Stage-1 跑通 baseline。
5. **长尾** ✅ **先只用原始 Messidor 跑通 baseline**,暂不接 MAPLES-DR。

> 以上已固化进代码与配置;实现见 `drnet/`、`scripts/`、`configs/`。
